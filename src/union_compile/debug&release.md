### generate debug and release procedure

+ main.f90
```fortran
program p
    integer::x
    x=1
    x=2
    x=3
end program p
```

***release***
```shell
$> gcc main.f90 -o main
```

***debug***
+ add flag `g`
```shell
$> gcc -g main.f90 -o main
```
